Shelly 2023

This is a font based on the calligraphy in Shelly Kim's Digital Hand Lettering and Modern Calligraphy. It's a loose adaptation; the book shows a more cursive hand, with all the letters joined up. That style has its good points, but it's not what I wanted to do.

Here's some sample text: The quick brown fox jumps over the lazy dog. I have a longer sample, too. And, if your browser doesn't show you the sample text in this font, you can see the PDF.

As of release 0.2 (15 July 2023), this font supports all of Latin-1 (that is, the characters used in Western European languages), and has a few ligatures. I intend to add more accented characters, and maybe some more ligatures. I have kerning for the plain ASCII letters, but not for the accented ones; a future release of the font will probably add that, which will change the layout of documents using the font.

Download

  • TrueType: download to your computer and install it as a font.
  • WOFF: copy it into a Website and reference it with CSS, so you can have this font in an Web page.
